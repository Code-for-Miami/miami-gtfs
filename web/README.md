### Bad Doggos

An interactive map of dangerous dogs in Austin

[Live Demo](https://bad-doggos.ideakeg.xyz)

### Development Instructions:

* Install dependencies: `yarn`
* Run a local server: `yarn start`

### Testing Instructions:

* Run: `yarn test`
