# Miami GTFS Data

#### Requirements
* Node 7.7+
* Docker and Docker Compose

#### Server
`cd server && docker-compose up`
After starting the server, send a get request to localhost:3600/import

#### Setup Web
`cd web && yarn`

#### Run Web
`cd web && yarn start`
